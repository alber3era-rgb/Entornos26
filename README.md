# Entornos26 - BuggyWebApp

Este proyecto es parte del **Reto 2** de la asignatura Entornos de Desarrollo. Se trata de una aplicación Java (BuggyWebApp) que presentaba una vulnerabilidad lógica en el procesamiento de formularios, la cual ha sido analizada y corregida.

## 🚀 Objetivos del Reto
- Configuración de un entorno de desarrollo profesional (**IntelliJ IDEA**).
- Gestión de versiones con **Git** y **GitHub** (Repositorio `Entornos26`).
- Uso del **Debugger** para la localización de errores (Partes 3 y 4).
- Documentación técnica de incidencias (**InformeTecnico.md**).
- Aplicación de buenas prácticas y licencias de software libre (**GPL-3.0**).

## 🛠️ Tecnologías utilizadas
- **Java** (JDK 17/21)
- **IntelliJ IDEA**
- **Git/GitHub**

## 📄 Contenido
- `src/`: Carpeta con el código fuente corregido.
- `InformeTecnico.md`: Informe con las evidencias del debugger y análisis del fallo.
- `LICENSE`: Archivo de licencia GNU GPL v3.0.
